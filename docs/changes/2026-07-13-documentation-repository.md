# 변경 기록: 문서 전용 저장소 초기화

- 날짜: 2026-07-13
- 변경 유형: 운영

## 변경 전

공개 저장소가 비어 있어 개발 과정과 평가 결과를 소스 코드와 분리해 공유할 기준이 없었습니다.

## 변경 후

개발, 변경, 평가와 설계 결정을 서로 구분한 문서 구조를 만들었습니다. 문서 작성 양식과 공개 정책을 추가했고, 허용된 문서 형식 외 파일을 차단하는 저장소 검사를 구성했습니다.

## 호환성 영향

제품 실행과 비공개 소스 저장소에는 변화가 없습니다. 이 저장소에는 공개 가능한 문서만 별도로 작성합니다.

## 검증

- 추적 파일 확장자 허용 목록 검사
- 금지된 소스 확장자를 추가했을 때 검사 실패 여부 확인
- 문서 내부 링크와 디렉터리 구조 수동 확인
