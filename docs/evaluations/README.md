# 분석 및 평가

기능과 구조 변경을 어떤 기준으로 검증했고 무엇을 관찰했는지 기록합니다.

평가는 성공 결과뿐 아니라 실패 사례, 자료 부족, 환경 제약과 아직 검증하지 못한 항목을 함께 밝힙니다. 개인 자료가 필요했던 평가는 원문 대신 익명화된 조건과 집계 결과만 사용합니다.

새 문서는 `YYYY-MM-DD-주제.md` 형식으로 만들고 [평가 기록 양식](../templates/EVALUATION_RECORD.md)을 사용합니다.
