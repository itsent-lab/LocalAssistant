# 개발 기록

완료된 개발 단위의 목적, 범위와 구현 결과를 기록합니다. 소스 코드나 내부 설정은 포함하지 않습니다.

각 기록은 다음을 포함합니다.

- 해결하려던 실제 문제
- 변경 범위와 제외 범위
- 사용자 관점의 동작 변화
- 완료 조건
- 검증 문서 링크

새 문서는 `YYYY-MM-DD-주제.md` 형식으로 만들고 [개발 기록 양식](../templates/DEVELOPMENT_RECORD.md)을 사용합니다.
