# LocalAssistant 개발 문서

이 저장소는 LocalAssistant의 **개발 과정과 검증 결과를 공개하기 위한 문서 전용 저장소**입니다.

애플리케이션 소스 코드, 실행 설정, 개인 데이터, 인증 정보와 로컬 운영 상태는 이 저장소에 포함하지 않습니다. 문서는 실제로 구현하거나 검증한 내용만 기록하며, 계획이나 가설은 현재 동작과 명확히 구분합니다.

개발 리포트의 원문은 비공개 소스 저장소에도 함께 보존합니다. 이 GitHub 저장소는 그중 공개 정책을 통과한 문서만 단방향으로 게시하는 공개 문서 저장소이며, 소스 저장소를 미러링하지 않습니다.

## 문서 범위

- [개발 기록](docs/development/README.md): 구현 목적, 범위와 완료 조건
- [변경 기록](docs/changes/README.md): 사용자에게 보이는 변경과 호환성 영향
- [분석 및 평가](docs/evaluations/README.md): 검증 방법, 결과, 한계와 후속 판단
- [설계 결정](docs/decisions/README.md): 중요한 선택의 배경, 대안과 결과
- [공개 정책](docs/PUBLICATION_POLICY.md): 포함·제외 기준과 보안 원칙

## 현재 상태

문서 저장소의 초기 구조와 문서 전용 검사가 마련되었습니다. 현재 공개된 최신 기록은 다음과 같습니다.

- [항목별 자료 수집과 Wiki Browser 편집](docs/development/2026-07-13-item-collection-and-wiki-browser.md)
- [현재 작업분 검증 결과](docs/evaluations/2026-07-13-working-tree-validation.md)

## 원칙

1. 소스 코드와 실행 가능한 스크립트를 게시하지 않습니다.
2. 개인·가족 자료, 로컬 경로, 비밀 값과 원문 대화는 게시하지 않습니다.
3. 수정 내용에는 검증 방법과 관찰된 결과를 함께 기록합니다.
4. 실패와 한계도 성공 결과와 같은 수준으로 남깁니다.
5. 미래 계획은 완료된 기능처럼 표현하지 않습니다.
